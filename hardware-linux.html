Linux Hardware Detection Commands

1. sudo lshw

This command provides detailed information about the hardware configuration of the system, including CPU, memory, storage, network interfaces, and more. It is often used for comprehensive system analysis.

        sudo lshw

2. lscpu

The lscpu command displays information about the CPU architecture, including the number of CPUs, cores, threads, model name, clock speed, and cache sizes.

        lscpu

3. lsblk

The lsblk command lists information about block devices, such as hard drives, SSDs, partitions, and USB devices. It displays device names, sizes, types, and mount points.

        lsblk

4. lspci

The lspci command shows information about PCI (Peripheral Component Interconnect) devices on the system, such as graphics cards, network adapters, and sound cards.

        lspci

5. lsusb

The lsusb command displays information about USB devices connected to the system, such as printers, storage devices, and other peripherals.

        lsusb

6. free

The free command provides a summary of the system's memory usage, including total, used, and free memory, as well as swap space utilization.

        free -h

7. df

The df command reports the amount of disk space used and available on the file system. The -h option shows the output in a human-readable format (e.g., GB or MB).

        df -h

8. hwinfo

The hwinfo command provides detailed information about the hardware components, including CPU, memory, network adapters, video cards, and more. It is similar to lshw, but with additional details.

        sudo hwinfo

9. dmesg

The dmesg command prints the kernel ring buffer, which includes detailed system messages related to hardware events, such as device initialization and driver loading.

        dmesg

10. inxi

The inxi command is a powerful tool for displaying comprehensive hardware information in a user-friendly format. It shows information about the system's CPU, memory, storage, network, and other peripherals.

        inxi -Fxz

11. lshw -short

This is a short version of the lshw command that provides a condensed summary of hardware information, including details about CPU, memory, and devices.

        sudo lshw -short

12. watch -n 1 lspci

The watch command allows you to continuously monitor the output of another command. In this case, it repeatedly runs lspci every second, allowing you to track changes in PCI devices.

        watch -n 1 lspci

13. cat /proc/cpuinfo

This command displays detailed information about the CPU, including the number of cores, clock speed, architecture, and more. It retrieves the information from the /proc/cpuinfo file.

        cat /proc/cpuinfo

14. cat /proc/meminfo

The cat /proc/meminfo command displays detailed information about system memory, including total memory, free memory, buffers, and swap usage.

        cat /proc/meminfo

15. uname -a

The uname -a command prints information about the kernel, including the kernel version, system architecture, and operating system.

        uname -a

16. dmesg | grep -i error

This command filters the dmesg output to show only error messages, helping you quickly identify potential hardware issues or problems with device drivers.

        dmesg | grep -i error
